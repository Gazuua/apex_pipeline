# architecture_comparison.html v0.6.1 데이터 갱신

**브랜치**: `feature/benchmark-html-v061`
**백로그**: BACKLOG-203
**상태**: 완료

## 변경 요약

gh-pages로 배포되는 `docs/apex_core/benchmark/architecture_comparison.html`을
v0.5.10.0 (i5-9300H 4C/8T) 데이터에서 v0.6.1.0 (i7-14700 20C/28T) 데이터로 전면 갱신.

## 주요 변경

| 항목 | v0.5.10 (기존) | v0.6.1 (갱신) |
|------|---------------|---------------|
| 환경 | i5-9300H 4C/8T | i7-14700 20C/28T (P8+E12) |
| 비교 모델 | 2개 (Per-Core, Shared) | 3개 (Per-Core, Shared, Lock-Free) |
| 워커 데이터포인트 | 6개 (1,2,3,4,8,16) | 8개 (1,2,3,4,8,16,20,28) |
| Per-core 피크 | 2.11M (16w) | 7.65M (8w) |
| Shared 피크 | 0.77M (16w, 정체) | 2.34M (4w, 이후 하락) |
| 최대 배율 | 2.1x (4w) | 5.3x (28w) |

## 분석 텍스트 갱신

- P/E 하이브리드 코어 아키텍처에 대한 해설 추가 (8 P-core에서 피크, E-core 영역 완만 하락)
- Lock-Free 모델 분석 카드 신설 — CAS 실패 재시도의 고경합 역효과 설명
- v0.5.10 → v0.6.1 비교 분석 카드 추가 — 코어 수 증가에 따른 격차 발산 해설
- 차트에 P-core 경계(8w) 어노테이션, 배율 차트에 2x/4x 기준선 추가
