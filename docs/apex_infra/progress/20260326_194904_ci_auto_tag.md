- **백로그**: BACKLOG-178

## 내용

`release-tag` 잡이 main 머지 시 `VERSION` 파일 기반으로 `vX.Y.Z.N` 태그를 자동 생성한다.

- CI 전체 통과(빌드+테스트+포맷) 조건에서만 태깅
- 4번째 자리(소)는 CI auto-tag가 자동 증가 관리
- 상위 3자리는 루트 `VERSION` 파일에서 읽음
