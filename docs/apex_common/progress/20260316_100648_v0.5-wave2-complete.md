### 구현 상세

#### Plan 0: WireHeader v2 + Transport concept + Kafka Envelope
- WireHeader에 `msg_id` (u32) 필드 추가
- Transport concept 정의 (프로토콜 추상화)
- Kafka Envelope 직렬화/역직렬화

#### Plan 1: Gateway MVP — 34 신규 파일
- TLS 핸드셰이크 (OpenSSL 기반)
- JWT 검증 미들웨어
- 서비스 라우팅 (메시지 타입 → 서비스 매핑)
- Pub/Sub 브로드캐스트 지원

#### Plan 2: Rate Limiting
- 3계층 Sliding Window Counter 구현
  - Per-connection, Per-user, Global 제한
- 설정 가능한 윈도우 크기 및 임계값

#### Plan 3: Auth Service
- JWT 발급 (Access Token + Refresh Token)
- Redis 세션 저장소
- Refresh Token 갱신 플로우
- bcrypt 비밀번호 해싱 번들

#### Plan 4: Chat Service
- 방(Room) 관리 (생성/참여/퇴장)
- Redis Pub/Sub 기반 브로드캐스트
- 채팅 히스토리 저장/조회

#### Plan 5: E2E 테스트 프레임워크
- 통합 테스트 인프라 구축
- docker-compose Redis 4분리 (Auth/Chat/Session/PubSub)
